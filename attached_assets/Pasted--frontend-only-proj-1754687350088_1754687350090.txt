Новые роуты

Dashboard компоненты (compact)

ProjectHeader: название, домен, кнопки: «Новое обогащение», «Импорт CSV», «История».

LastRunCard: статус, дата, задачи (чипы), counters (added/rejected), кнопка «Продолжить ревью» если Draft.

KPIGrid (4–6 тайлов): Orphans, Avg Depth, Links Added, 404 Fixed, Anchor Profile. Каждый показывает Δ от предпредыдущего run.

DataSourceCard: имя CSV, строки, дата импорта, «Переимпортировать».

Run Workspace (один экран, без попапов)

Глобальные параметры наверху (Max N, порядок приоритетов drag&drop, minGap, exact%, stop-anchors, Old links, Broken, HTML attrs, Scope).

Быстрые задачи — чекбоксы с мини-полями:

Хабы по префиксу: загрузка CSV prefix, subprefix?, hub_url, топология (звезда/кольцо/колесо), «ограничить внутри (под)префикса».

Money: паттерн URL (regex).

Похожие: префикс(ы), k=1..3.

Глубинные: minDepth, доноры ≤3.

Свежие: daysFresh, linksPerDonor.

Сироты: область (сайт/префикс).

Чистка 404: политика Delete/Replace/Ignore.

Перегенерация/обогащение: Enrich/Regenerate.

Preflight строкой: «Ожидаем ≈ X ссылок (Хаб ~A, Коммерция ~B, …)».

Кнопка “Запустить” → прогресс (эмуляция фаз) → статус Draft → сразу ниже Draft Review.

Draft Review таблица inline (фильтры по типам, чекбоксы «Одобрено», inline-diff). Кнопки «Одобрить видимые», «Экспорт CSV», «Применить».

Run History страница

Таблица: дата/время, статус, задачи (чипы), counts (added/rejected), длительность.

Фильтры по статусу. Клик по строке → открывает соответствующий Draft/Result (тот же компонент, в read-only, в этой странице).
